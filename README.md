# dvrepair
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

DV tape repair tool - conjoin multiple captures of the same tape

## About dvrepair

`dvrepair` is build to work against specific problem - media read errors. When MiniDV or any DV camera may suffer from read/write errors (like bad tape mechanism, or just tape degradation), the output record will result in the errors, that can be detected even **after the dumping the records onto PC**. 

This feature of DV digital media format **is** the thing that allows to take 2-6 or more snippets of tape, and repair it to then, eventually, get an fully recovered record

## Usage

```
dvrepair --main-stream main.avi 1.avi 2.avi 3.avi 4.avi 5.avi
```
